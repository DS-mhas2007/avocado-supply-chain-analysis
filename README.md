# 🥑 Avocado Toast Supply Chain Analysis

## 📌 Overview
This project analyzes the supply chain of key ingredients used in avocado toast in the UK using the Open Food Facts dataset.

## 🎯 Objectives
- Identify the most common country of origin for:
  - Avocados
  - Olive Oil
  - Sourdough Bread

## 🛠 Tools Used
- Python
- Pandas
- Data Cleaning Techniques
- Regex
- Explode (for handling multi-value fields)

## 🔍 Key Steps
- Data cleaning and preprocessing
- Filtering relevant categories
- Handling missing values
- Splitting and expanding multi-value columns
- Aggregation to find most frequent origins

## 📊 Results
- 🥑 Avocado: Peru
- 🫒 Olive Oil: Greece
- 🍞 Sourdough: United Kingdom

## 💡 Key Insights
- Avocados are mostly imported
- Olive oil mainly comes from Mediterranean countries
- Sourdough is locally produced
